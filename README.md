# AI-Text-Generator-using-Markov-Chains
AI Text Generator using Markov Chains
## 📌 Description
This project generates text using Markov Chains implemented with Markovify library. The model learns patterns from Shakespeare's text corpus and generates new sentences.

## ⚙️ Features
- Text preprocessing
- Markov Chain probabilistic model
- Sentence generation
- Short text generation

## 🧠 Algorithm
Markov Chain (State-based probabilistic text generation)

## 📊 Dataset
Shakespeare Tragedies from NLTK Gutenberg Corpus

## 🚀 Output
Generated synthetic text similar to Shakespearean style
